# Create a Page

## Technologies

- NodeJS (ExpressJS)
- MongoDB
- Redis
- PM2
- Elasticsearch

## Setup

1. Clone file env and rename to .env. Then add configuration to .env file
2. Install packages

```
$ npm install
```

3. Deploy

```
$ npm run deploy
```

4. Dev

```
$ npm run dev
```
